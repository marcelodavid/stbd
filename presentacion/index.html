Sistemas de Telemedida

Cuentan con una gran cantidad de puntos de medición, deben permitir el registro y extracción de datos y facilitar el servicio de consulta.

Elementos del Sistema

JUSTIFICACIÓN

La inexistencia de un sistema capaz de monitorear el consumo y detectar los inconvenientes citados, produce un déficit en el control energético con un tiempo muy prolongado de reacción en la resolución de problemas, por parte de la entidad encargada del suministro eléctrico.

FINALIDAD

Este proyecto tiene como finalidad proporcionar a la empresa proveedora de electricidad y a sus abonados, una herramienta de monitoreo, con el propósito de acceder en tiempo real al consumo y la demanda de energía eléctrica en cada hogar.

OBJETIVOS

Objetivo General

Desarrollar un sistema de telemedida que facilite el control de la demanda y el consumo de la energía eléctrica de usuarios domiciliarios.

Objetivo Específico

Diseñar un dispositivo de lectura inalámbrico y un dispositivo concentrador(gateway) para recolectar los datos de un medidor eléctrico inteligente

Objetivo Específico

Diseñar la red de comunicacion entre los dispositivos de lectura y el dispositivo concentrador

Objetivo Específico

Desarrollar el software necesario para realizar la telemedición del consumo eléctrico a través de una interfaz web.

Objetivo Específico

Realizar pruebas de campo del prototipo propuesto

Objetivo Específico

Evaluar la viabilidad económica del proyecto

Recomendaciones

  • Reemplazar el modulo RF Xbee serie 2 por el módulo NRF24l01+.
  • Emplear un arduino nano con el modulo inalambrico NRF24l01+.
  • Intercambiar la fuente de alimentación del prototipo por paneles solares.
  • Actualizar la aplicación web con mayores funcionalidades.
  • Emplear comunicación asincrona entre los nodos y el dispositivo concentrador
